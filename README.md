# 지구, 태양 주위를 떠도는 여정! 🌍

Streamlit으로 만든 웹 애플리케이션입니다.

## 설치 방법

1. 가상환경 생성 (선택사항)
```bash
python -m venv venv
venv\Scripts\activate  # Windows
```

2. 패키지 설치
```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run app.py
```

브라우저에서 자동으로 열립니다. (기본: http://localhost:8501)

## 기능

- 기능 1: 텍스트 입력
- 기능 2: 버튼 인터랙션

## 개발

앱을 수정하고 싶다면 `app.py` 파일을 편집하세요.
